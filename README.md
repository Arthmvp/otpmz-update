## Suporte: ![Badge Nvidia](https://img.shields.io/badge/GPU's-Nvidia-green) ![Badge Intel](https://img.shields.io/badge/CPU's%20&%20GPU's%20(Integrado)-Intel-blue) ![Badge AMD](https://img.shields.io/badge/CPU's%20e%20GPU's%20(Dedicado%20&%20Integrado)-AMD-darkred)

**Jotah Store Optimizer** é uma suíte completa de otimização para Windows, desenvolvida para extrair o máximo de desempenho do seu hardware, focando em estabilidade para jogos competitivos e agilidade para produtividade.

Com uma interface moderna baseada em *Glassmorphism*, o software oferece ferramentas avançadas para limpeza de sistema, ajustes de kernel, otimização de rede e configurações específicas para jogos como FiveM.

## ✨ Funcionalidades Principais

### 🧠 Otimização Inteligente
Utiliza algoritmos para detectar automaticamente a arquitetura do seu processador e quantidade de RAM, aplicando um perfil de otimização cirúrgico que ajusta prioridades de threads e gerenciamento de memória.

### 🧹 Limpeza e Manutenção
- **Arquivos Temporários:** Remoção segura de lixo do sistema.
- **Cache & Logs:** Limpeza profunda de registros antigos do Windows.
- **Rede:** Flush DNS e otimização de parâmetros TCP para redução de latência.

### ⚡ Performance de Hardware
- **CPU:** Ajustes de prioridade (AMD/Intel), Tweaks de BCD e Mitigação de Processos.
- **RAM:** Gerenciamento inteligente de memória e controle de compactação (MMAgent).
- **Energia:** Integração com planos de energia de alta performance (ParkControl).

### 🎮 Otimização para Jogos
- **FiveM:** Configurações de FPS personalizadas e limpeza de cache específica para corrigir erros comuns (crashes).
- **Game Mode:** Desativação de serviços desnecessários (Game Bar, DVR) que consomem recursos em segundo plano.

### 🛠️ Ajustes do Windows (Debloat)
- **Serviços:** Desativação de serviços não essenciais e telemetria.
- **Responsividade:** Ajustes no registro para tornar a interface e inputs mais ágeis.
- **Periféricos:** Correção de aceleração do mouse (Mouse Fix 1:1).

> **Nota:** A versão **Premium (V2)** desbloqueia recursos exclusivos como a IA de detecção de hardware, ajustes profundos de Kernel e suporte prioritário.

## ⚠️ Aviso Legal

Este software realiza alterações no registro do Windows e configurações de sistema. Embora testado exaustivamente para garantir estabilidade e segurança, o uso é por sua conta e risco.

Recomendamos criar um **Ponto de Restauração** do Windows antes de aplicar otimizações profundas.
